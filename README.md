# Initial Setup

> rename <b><i>/api/config/config_sample.env</i></b> to <b><i>/api/config/config.env</i></b> and add your values.

# Install Dependencies

```
npm i
```

# How to Run

```
# development mode
npm run dev

# production mode
npm start
```
